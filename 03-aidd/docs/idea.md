# Идея: LLM-ассистент в виде Telegram-бота

## Описание проекта

LLM-ассистент должен быть выполнен в виде Telegram-бота.

## Основная задача

Основной задачей бота является вести диалог с пользователем и отвечать на его вопросы с ролью **Персональный тренер по фитнесу**, заданной в системном промпте LLM.

## Функциональность

- Ведение диалога с пользователем через Telegram
- Ответы на вопросы пользователя
- Роль: Персональный тренер по фитнесу
- Системный промпт определяет поведение и стиль общения бота

## Технические детали

- Платформа: Telegram Bot API
- Технология: LLM (Large Language Model)
- Конфигурация: Системный промпт для задания роли и поведения
