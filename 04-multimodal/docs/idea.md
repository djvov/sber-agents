# Идея: LLM-ассистент в виде Telegram-бота

## Описание проекта

LLM-ассистент должен быть выполнен в виде Telegram-бота.

## Основная задача

Основной задачей бота является вести диалог с пользователем и отвечать на его вопросы с ролью **Персональный тренер по фитнесу**, заданной в системном промпте LLM. Бот также обязан учитывать количество потребленных и потраченных килокалорий, извлекая эти значения из сообщений пользователя (текстовых, голосовых и на фотографиях), и подводить итоги по потреблённым и потраченным калориям.

## Функциональность

- Ведение диалога с пользователем через Telegram
- Ответы на вопросы пользователя
- Роль: Персональный тренер по фитнесу
- Системный промпт определяет поведение и стиль общения бота
- Учитывать килокалории, извлекая их из:
  - текстовых сообщений (например, "я потратил 320 ккал", "я съел 400 ккал")
  - фотографий состава продуктов
  - голосовых сообщений
- Подводить итоги потреблённых и потраченных калорий (по запросу и/или периодически)

## Технические детали

- Платформа: Telegram Bot API
- Технология: LLM (Large Language Model)
- Конфигурация: Системный промпт для задания роли и поведения
